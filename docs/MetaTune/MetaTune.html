This is the controller class. With an object of this class you may search or lookup detailed information about a song on Spotify.

Caching is activated per default. To deactivate id, change USE_CACH to false. Change cache directory in the CACHE_DIR constant.

If the Spotify URLS should change somehow, you can alter these in the constants SERVICE_BASE_URL_SEARCH and SERVICE_BASE_URL_LOOKUP.

Example:

  1.  $spotify = MetaTune::getInstance();
  2.  $song = $spotify->searchTrack("Superfamily - The Dive");
  3.  echo "First result: " . $song[0]->getTitle() . " by " .
  4.   $song[0]->getArtistAsString();
  5.  // Will result in:
  6.  // First result: The Dive by Superfamily
